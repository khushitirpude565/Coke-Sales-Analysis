# Coke Sales Analysis using Excel

## Project Objective
The objective of this project is to analyze **:contentReference[oaicite:0]{index=0}** sales data across U.S. retailers and regions using Microsoft Excel.  
The analysis focuses on understanding sales performance, units sold, and operating margin to identify key trends and improvement opportunities.

---

## Dataset Overview
The dataset contains transactional sales data with the following key attributes:
- Date
- Retailer Name
- Region
- State
- Units Sold
- Price per Unit
- Total Sales
- Operating Margin

---

## Tools Used
- Microsoft Excel
- Pivot Tables
- Excel Formulas
- Bar Charts

---

## Data Preparation
- Reviewed and understood the dataset structure
- Standardized column names
- Corrected data types for numeric and date fields
- Verified calculated metrics such as total sales and operating profit

---

## Analysis Performed
- **Sales Analysis by Region** using pivot tables
- **Units Sold Analysis by Retailer**
- **Operating Margin Analysis by Region** using average values
- **Time-based trend analysis** using Month-Year aggregation

---

## Visualizations
- Bar chart showing total sales by region
- Bar chart comparing units sold by retailer
- Bar chart showing average operating margin by region

---

## Key Insights
- The West region contributes the highest total sales
- Retailer performance varies significantly in terms of units sold
- Operating margins differ across regions, indicating profitability variation
- Sales patterns show regional and seasonal trends

---

## Conclusion
This analysis helps identify high-performing regions and retailers while highlighting areas where pricing and margin optimization can improve overall profitability.
